# RK Visualizer

Runge-Kutta metoder er numeriske metoder som brukes for å løse differensialligninger.
Dette python-prosjekt visualiserer noen av dem :D

## Kjøre
```bash
python3 run_visualizer.py
```
Da starter en lokal server pa `http://127.0.0.1:8000`.

Parseren støtter blant annet:

- `sin`, `cos`, `tan`
- `exp`, `log`, `log10`, `sqrt`
- `abs`, `min`, `max`
- `pi`, `e`
